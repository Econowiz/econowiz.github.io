## 🚀 Quick Start

### Prerequisites
- 📦 Node.js 18+ 
- 🧶 pnpm (recommended) or npm

### Installation

```bash
# Clone the repository
git clone https://github.com/econowiz/portfolio-card-fra.git

# Navigate to project directory
cd portfolio-card-fra

# Install dependencies
pnpm install

# Start development server
pnpm run dev
```

🎉 **That's it!** Open [http://localhost:5173](http://localhost:5173) to view your portfolio.
